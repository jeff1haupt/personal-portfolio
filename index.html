About

What I do

I am a student and a teacher. With every project I build, I am learning something new. With every student I mentor, I am learning something new. But, with each project I work with a student on is a chance for me to pass on my knowledge.

I apply my knowledge in the areas of Java, Spring Boot, and MySQL to assist students who are a part of a community college program learning backend development using these tools as part of a coding bootcamp. As a curriculum specialist, I get to assist in the development of reviewing and authoring new project and items to this course curriculum.

As a freelance developer, I am gaining additional skills in React and React Native to build complete applications for small businesses utilizing Spring Boot REST API development with React to connect to the API. This allows me the ability to build out Single Page Web Applications to small businesses that are scaled to fit their needs.

Tools I use

  • Java and Spring Boot
  • React and React Native
  • JavaScript and JQuery
  • HTML and CSS
  • MySQL and MongoDB
